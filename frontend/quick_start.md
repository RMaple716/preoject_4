# 地图调试系统 - 快速开始

## 已完成的集成

✅ 调试版地图组件已集成到 `navigationinfo.tsx`
✅ 点击"查看地图"按钮时会显示带有调试功能的地图Modal
✅ 地图Modal标题栏包含"运行诊断"按钮

## 如何使用

### 1. 打开行程详情页

访问任意行程详情页面（例如：`/itinerary/{id}`）

### 2. 查看交通信息

在行程详情中找到交通信息卡片，点击"查看地图"按钮

### 3. 运行诊断

在打开的地图Modal中，点击右上角的"运行诊断"按钮

### 4. 查看诊断报告

打开浏览器控制台（F12），查看详细的诊断报告

## 诊断报告内容

诊断报告会检查以下内容：

1. **高德地图脚本** - 是否正确加载
2. **地图容器** - 是否存在且尺寸有效
3. **地图实例** - 是否成功创建
4. **路线数据** - 是否包含有效的polyline
5. **API Key** - 是否有效

## 常见问题

### 问题：地图不显示

**解决方案**：
1. 点击"运行诊断"按钮
2. 查看诊断报告中的❌标记
3. 根据修复建议解决问题

### 问题：看不到日志

**解决方案**：
1. 确认浏览器控制台已打开（F12）
2. 确认在开发环境运行
3. 检查控制台过滤设置

### 问题：诊断报告显示错误

**解决方案**：
1. 查看诊断报告中的"修复建议"部分
2. 按照建议逐一修复
3. 重新运行诊断确认问题已解决
